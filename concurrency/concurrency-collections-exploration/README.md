# concurrency-collections-exploration
Focus on: ConcurrentHashMap, CopyOnWriteArrayList, Queue-Concurrent-Collections

## Topics
 * maps       – mainly ConcurrentHashMap (most common & performant)
 * lists      – CopyOnWriteArrayList + comparison with synchronized list
 * queues     – ConcurrentLinkedQueue, ArrayBlockingQueue, LinkedBlockingQueue

## Stack
- Java 17 
- JUnit 5 (tests)

## Run
```
mvn clean package
```
